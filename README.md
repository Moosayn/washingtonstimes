# WashingtonsTimes News Website

A professional news website with a custom CMS for managing and publishing articles.

## Features

- Professional multi-column news layout
- Article management system (create, edit, delete)
- SEO-friendly URLs
- Category organization
- Responsive design
- Image upload support
- Clean, modern interface

## Setup

1. Install dependencies:
   ```
   npm install
   ```

2. Start the server:
   ```
   npm start
   ```

3. Visit http://localhost:3000

## Admin Panel

Access the admin panel at http://localhost:3000/admin to:
- Create new articles
- Edit existing articles
- Manage categories
- Upload images

## Project Structure

- `/public` - Static assets (CSS, JS, images)
- `/views` - EJS templates
- `/data` - Article storage
- `/uploads` - User uploaded images
- `server.js` - Main application server
